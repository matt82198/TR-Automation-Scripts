# Configuration Directory

This directory contains configuration files and application state for the TR Automation Scripts.

## Files

### Gmail OAuth2 Credentials (REQUIRED)
- **`gmail_credentials.json`** - OAuth2 credentials from Google Cloud Console
  - **Status**: Required for email functionality
  - **Committed**: NO (in .gitignore)
  - **Setup**: See `/docs/EMAIL_SETUP.md`

- **`gmail_token.json`** - OAuth2 access token (auto-generated)
  - **Status**: Auto-generated on first run
  - **Committed**: NO (in .gitignore)
  - **Note**: Will be refreshed automatically when expired

### Product Mappings
- **`sku_mapping.csv`** - Maps Squarespace product names to QuickBooks item names
  - **Status**: Required for accurate product mapping
  - **Committed**: YES (example mappings)
  - **Format**: `SquarespaceProductName,QuickBooksItem`
  - **Note**: See `/docs/SKU_MAPPING_GUIDE.md` for details

### Application State
- **`import_log.csv`** - Tracks imported orders to prevent duplicates
  - **Status**: Auto-generated by script
  - **Committed**: NO (in .gitignore)
  - **Format**: `order_number,date_imported,iif_file`
  - **Note**: Delete this file to re-import orders

## Setup Instructions

1. **Gmail OAuth2 Setup** (Required for email)
   ```bash
   # Follow the guide to download credentials
   cp /path/to/downloaded/credentials.json config/gmail_credentials.json
   ```
   See: `/docs/EMAIL_SETUP.md`

2. **Product Mapping** (Optional but recommended)
   - Edit `config/sku_mapping.csv` to add your mappings
   - See: `/docs/SKU_MAPPING_GUIDE.md`

3. **Import Log** (Auto-created)
   - Created automatically on first import
   - To reset: `echo "order_number,date_imported,iif_file" > config/import_log.csv`
